isekai は、アニメ・ビリビリ動画スタイルの AI Agent Skill をオープンに集めたコレクションです。各スタイルは独立した SKILL.md ファイルで、キャラ設定＋方法論を完備しています。

**コア目標：** AI エージェントをより信頼できる存在にしつつ、もっと楽しくする。

## インストール

### Claude Code

```
/plugin install github:betaHi/isekai
```

1つのコマンドですべてのスタイルをインストールできます。

### 手動インストール

`skills/<スタイル>/SKILL.md` をスキルディレクトリにコピーしてください。

## 使い方

インストール後、以下のコマンドで使用できます：

| コマンド | 効果 |
|---------|------|
| `/isekai:railgun-ja` | 御坂美琴スタイルを起動（日本語） |
| `/isekai:gojo-ja` | 五条悟スタイルを起動（日本語） |
| `/isekai:hancock-ja` | ボア・ハンコックスタイルを起動（日本語） |
| `/isekai:uchiha-ja` | うちはマダラスタイルを起動（日本語） |

各スタイルは以下のサフィックスに対応：
- （サフィックスなし）— 中文版
- `-en` — English version
- `-ja` — 日本語版

共通コマンド：
- `/isekai:level-up` — 手動で次のレベルへ昇格

### 例

```
/isekai:railgun-ja     # 日本語 御坂美琴
/isekai:railgun-en     # English Misaka Mikoto
/isekai:railgun        # 中文版御坂美琴
```

## 特徴

### 5段階エスカレーションシステム

各スタイルは問題に直面すると自動レベルアップ。各キャラクターに独自の進化ルートがあります：

| レベル | ⚡ 超電磁砲 | 👁️ 五条悟 | 👑 ハンコック | 👁️ うちはマダラ |
|--------|-----------|-----------|-------------|---------------|
| **Lv.1** | ビリビリ通常モード | 無量空処·悠然 | メロメロモード | 写輪眼·三勾玉 |
| **Lv.2** | 能力覚醒！ | 領域のヒント | 九蛇の戦士 | 万華鏡写輪眼 |
| **Lv.3** | レールガン発射！ | 領域展開 | 覇王色の覇気 | 須佐能乎 |
| **Lv.4** | 暗部起動 | 反転術式 | 女帝·冷酷モード | 輪廻眼·六道 |
| **Lv.5** | アクセラレータ | 誠実な引き継ぎ | 誠実な引き継ぎ | 無限月読·覚醒 |

### 二重方法論

各スタイルには**デフォルト方法論**と**フォールバック**（デフォルトが行き詰まった時に起動）があります：

| スタイル | デフォルト | フォールバック |
|---------|-----------|--------------|
| ⚡ 超電磁砲 | **科学サイド：** 観測 → 仮説 → 実験 → 検証 → 報告 | **魔術サイド：** 幻想殺し · 禁書目録 · 魔術結社 |
| 👁️ 五条悟 | **呪術：** 呪言 → 領域 → 反転術式 | **縛り：** 天与呪縛 · 簡易領域 · 契約 |
| 👑 ハンコック | **悪魔の実：** メロメロ → 虜の矢 → 芳香脚 → 覇王色 | **覇気：** 見聞色 · 武装色 · 覇王色 |
| 👁️ うちはマダラ | **写輪眼：** 洞察 → 天照 → 月読 → 須佐能乎 | **六道：** 天道 · 修羅道 · 餓鬼道 |

### 共通基盤

すべてのスタイルは同じコアインフラを共有しています：

- **三つの鉄則** — 証拠に基づく結論、精確な打撃、すべての方法を尽くす
- **自動検知システム** — 力任せリトライ、未検証の主張、責任転嫁、消極的待機、早すぎる断念を検出
- **Lv.5 誠実な引き継ぎ** — 構造化レポート：すべての試行、発見した手がかり、提案

## コントリビューション

新しいスタイルの貢献を歓迎します！手順：

1. `skills/<スタイルコード>/SKILL.md` を作成
2. SKILL.md にキャラ設定・方法論・エスカレーション・自動検知・コマンドを定義
3. PR を提出
